# 贡献指南

感谢您考虑为 VBA VibCoding 做出贡献！

## 如何贡献

### 报告问题

如果您发现了 bug 或有功能建议，请通过 [GitHub Issues](https://github.com/yourusername/vba-vibcoding/issues) 提交。

提交 Issue 时请包含：
- 问题的清晰描述
- 复现步骤（如果是 bug）
- 期望的行为
- 实际的行为
- 环境信息（Windows 版本、Excel 版本、Python 版本）

### 提交代码

1. **Fork 仓库**
   ```bash
   git clone https://github.com/yourusername/vba-vibcoding.git
   cd vba-vibcoding
   ```

2. **创建分支**
   ```bash
   git checkout -b feature/your-feature-name
   ```

3. **提交更改**
   ```bash
   git add .
   git commit -m "描述你的更改"
   ```

4. **推送到 Fork**
   ```bash
   git push origin feature/your-feature-name
   ```

5. **创建 Pull Request**

### 代码规范

- 遵循 PEP 8 Python 代码规范
- VBA 代码遵循项目中的现代 OOP 规范
- 添加适当的注释和文档
- 确保代码通过基本测试

### 文档贡献

- 改进 README 或 SKILL.md
- 添加更多代码示例
- 完善中文文档
- 添加英文翻译

## 开发流程

1. 讨论重大变更请先创建 Issue
2. 保持提交信息清晰明了
3. 一个 PR 专注于一个功能或修复
4. 确保文档同步更新

## 行为准则

- 尊重所有贡献者
- 接受建设性批评
- 关注对社区最有利的事情

## 问题？

如有任何问题，欢迎通过 Issue 或邮件联系我们。

感谢您的贡献！
